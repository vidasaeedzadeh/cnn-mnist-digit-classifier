# CNN MNIST Digit Classifier

A simple Convolutional Neural Network (CNN) trained on the classic [MNIST digits dataset](https://en.wikipedia.org/wiki/MNIST_database) to classify handwritten digits (0–9).

The MNIST dataset consists of grayscale images of handwritten digits, each with dimensions of 28×28 pixels, with corresponding labels from 0 to 9.
